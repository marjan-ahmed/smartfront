# NexusFin Fintech Landing Page

A modern, engaging fintech landing page built with HTML, SASS, and JavaScript. Features include a search functionality, responsive design, interactive elements, and creative visual components.

## Features

- **Modern Fintech Design**: Professional financial technology landing page with clean aesthetics
- **Interactive Dashboard Preview**: Animated financial dashboard visualization
- **Advanced Search**: Search functionality with suggestions and live filtering
- **Responsive Design**: Fully responsive layout that works on all devices
- **Animated Elements**: Smooth animations and interactive hover effects
- **Professional Sections**: Features, solutions, testimonials, and statistics sections
- **AI-Powered Insights**: Demonstrates fintech AI capabilities

## Files

- `index.html` - Main HTML structure with semantic markup
- `styles.css` - Compiled SASS styles with modern design
- `script.js` - JavaScript functionality for interactive elements
- `sass/styles.scss` - SASS source file with variables and mixins

## Design Elements

- **Color Scheme**: Professional fintech colors with blue as primary (trust, security)
- **Typography**: Inter and Space Grotesk fonts for modern readability
- **Visual Hierarchy**: Clear information architecture with proper spacing
- **Interactive Components**: Buttons, cards, and navigation with hover effects
- **Data Visualization**: Dashboard preview with animated charts
- **Mobile-First**: Responsive design optimized for all screen sizes

## How to Use

1. Open `index.html` in a web browser
2. Explore the interactive elements:
   - Use the search bar to search financial tools
   - Hover over cards to see animations
   - Click navigation links to scroll to sections
   - Try the various call-to-action buttons

## Technical Implementation

- **SASS Variables**: Color scheme, typography, spacing, and breakpoints
- **CSS Grid & Flexbox**: Responsive layouts and component organization
- **JavaScript Interactions**: Search functionality, animations, and UI effects
- **Accessibility**: Proper semantic HTML and keyboard navigation support
- **Performance**: Optimized animations and efficient JavaScript

This landing page demonstrates modern web development techniques applied to the fintech industry with a focus on trust, security, and innovation.
